Pass input_file as the argument to the executable.

Run executable as follows: "Executable_name.exe input_file.txt"

Note: Make sure the full path to the input file is given. Else ensure the input file is in pwd.

Two output files are generated objPgmCode.txt and objectProgram.txt.
Any compile time errors will be displayed in objPgmCode.txt with explanation followed by !!!.
Final object code is available in objectProgram.txt
